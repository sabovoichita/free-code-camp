1. Comment Your JavaScript Code

Comments are lines of code that JavaScript will intentionally ignore. Comments are a great way to leave notes to yourself and to other oeple who will later need to figure out what that code does.

There are two ways to write comments in JavaScript:

1. Single line comment

Using // wll tell Java Script to ignore the reminder of the text on the current line. This is am in-line comment:

//This is an inline comment.

2. Multi-line comment

You can make a multi-line comment beggining with /* and ending with /*. This is a multi-line comment:

/*This is a
multi-line comment*/

Note:: As you write code, you should regularly add comments to clarify the function of parts of your code. Good commenting can helo communicate the intent of your code-both for others and for future self.

Exercise: Try creating one of each type of comment.
